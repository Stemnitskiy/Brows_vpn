# Xray-core Binary

Download Xray-core binary from https://github.com/XTLS/Xray-core/releases

Place the binary in this directory:
- Windows: xray.exe
- Linux: xray
- macOS: xray

For Windows 11 AMD64, download:
https://github.com/XTLS/Xray-core/releases/download/v1.8.6/Xray-windows-64.zip

Extract and place xray.exe in this directory.
