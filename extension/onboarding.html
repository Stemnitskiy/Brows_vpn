Настройка Brows VPN

Пошаговый мастер первого запуска

Добро пожаловать

Brows VPN состоит из расширения Chrome и локального Go-сервиса с Xray. Мастер проведёт через сборку, регистрацию Native Messaging и первое подключение.

  • Нужны: Go, Chrome, VLESS URL от провайдера
  • Все команды выполняются локально на вашем ПК
  • После настройки мастер можно открыть снова из Settings

Шаг 1 — Сборка Go-сервиса

Соберите browsvpn-proxy.exe в каталоге proxy-service.

  1. Откройте PowerShell в папке проекта
  2. Выполните команды ниже
  3. Убедитесь, что рядом лежит каталог xray-core
cd proxy-service
go build -o browsvpn-proxy.exe ./cmd

Шаг 2 — Регистрация Native Messaging

Свяжите Chrome с локальным прокси-сервисом через реестр Windows.

  1. Откройте proxy-service в проводнике
  2. Запустите setup_registry.bat (достаточно прав текущего пользователя)
  3. Дождитесь сообщения об успешной регистрации
cd proxy-service
.\setup_registry.bat

Шаг 3 — ID расширения

Разрешите этому расширению обращаться к Native Messaging host.

ID этого расширения:

…

Выполните в PowerShell из proxy-service (подставится ID автоматически):

powershell -File update_allowed_origins.ps1 -ExtensionId EXT_ID

После этого полностью перезапустите Chrome.

Шаг 4 — VLESS URL

Вставьте ссылку на сервер из панели провайдера.

Шаг 5 — Проверка и включение VPN

Убедитесь, что Go-сервис отвечает, затем включите VPN.

Готово

Базовая настройка завершена. Управление VPN — через иконку расширения.

  • Popup: включение/выключение, IP, профили
  • Settings: режимы, whitelist, правила, экспорт
  • Диагностика — в разделе Settings → Диагностика
