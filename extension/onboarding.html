Настройка Brows VPN

Пошаговый мастер первого запуска

Добро пожаловать

Brows VPN состоит из расширения Chrome и локального Go-сервиса с Xray. Мастер проведёт через установку native host и первое подключение.

  • Нужны: Chrome (или Chromium), VLESS URL от провайдера
  • Go 1.21+ — только если собираете Go-сервис из исходников (install.bat)
  • Расширение загружается из папки extension, не из корня проекта
  • После настройки мастер можно открыть снова из Settings → «Мастер»

Шаг 1 — Xray-core

Без xray.exe Go-сервис не сможет поднять VPN-тunnel.

  1. Скачайте Xray-windows-64.zip
  2. Распакуйте xray.exe в каталог proxy-service\xray-core\
  3. Рядом должны лежать geoip.dat и geosite.dat (уже в репозитории)
proxy-service\xray-core\xray.exe
cd proxy-service
Get-FileHash .\xray-core\xray.exe -Algorithm SHA256
Set-Content -Encoding ASCII .\xray-core\xray.exe.sha256 "<SHA256_FROM_PREVIOUS_COMMAND>"

Шаг 2 — Go-сервис и Native Messaging

Одна команда собирает browsvpn-proxy.exe, регистрирует native host с Extension ID из manifest.key и открывает chrome://extensions/.

  1. Откройте PowerShell в папке проекта
  2. Запустите install.bat в каталоге proxy-service
  3. Дождитесь сообщения «Install complete»
  4. Полностью перезапустите Chrome
cd proxy-service
.\install.bat
powershell -File .\scripts\check-env.ps1 -Fix -Release

Если browsvpn-proxy.exe уже собран, можно запустить .\install.ps1 без -Build.

Шаг 3 — Проверка расширения

Расширение уже загружено. Теперь нужно сверить его ID и убедиться, что native host зарегистрирован именно для этого ID.

  1. Сравните текущий ID с ожидаемым GitHub/unpacked ID ниже
  2. Если ID отличается, скопируйте repair command и перерегистрируйте native host
  3. После регистрации перезапустите Chrome и нажмите «Проверить снова»

Шаг 4 — VLESS URL

Вставьте ссылку на сервер из панели провайдера.

Шаг 5 — Проверка и включение VPN

Убедитесь, что Go-сервис отвечает, затем включите VPN.

Готово

Базовая настройка завершена. Управление VPN — через иконку расширения.

  • Popup: включение/выключение, IP, профили
  • Settings: режимы, whitelist, правила, экспорт
  • Диагностика — Settings → «Диагностика»
